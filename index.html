Key Concepts

What is Paging?

When RAM is full, the OS swaps pages to/from disk using virtual memory. Page replacement algorithms decide which page to evict.

Page Fault

Occurs when the requested page is not in RAM. The OS loads it from disk — a costly operation that slows execution.

Goal

Minimize page faults. Fewer faults = fewer disk accesses = faster program execution overall.

FIFO — First In, First Out
The page that has been in memory the longest is replaced first. Internally uses a queue data structure.
